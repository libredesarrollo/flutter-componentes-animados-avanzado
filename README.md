# Ejemplos Avanzados - Componentes y Animaciones en Flutter

Este proyecto contiene los ejemplos prácticos avanzados desarrollados en el libro **"Componentes y Animaciones en Flutter"**. Está diseñado para mostrar implementaciones del mundo real de conceptos complejos como scroll avanzado, animaciones de continuidad, interactividad táctil, integración de animaciones externas y accesibilidad.

## Contenido del Proyecto

El proyecto está dividido en secciones que corresponden a los capítulos finales del libro:

### 🌟 Capítulo 7: Slivers y Scroll Avanzado
- **Perfil Premium**: Una pantalla que demuestra el uso de `CustomScrollView`, `SliverAppBar` con efecto parallax, encabezados pegajosos (`SliverPersistentHeader`) y una grilla responsiva.

### 🎭 Capítulo 8: Hero y Transiciones
- **Catálogo Premium**: Implementación de animaciones `Hero` dinámicas y transiciones de pantalla personalizadas usando `PageRouteBuilder` para una navegación fluida.

### 👆 Capítulo 9: Gestures y Drag & Drop
- **Tablero Kanban**: Un ejemplo interactivo de arrastrar y soltar usando `LongPressDraggable` y `DragTarget`, con retroalimentación visual y háptica.

### 🎬 Capítulo 10: Animaciones Externas
- **Lottie y Rive**: Integración de animaciones JSON ligeras con Lottie y animaciones vectoriales interactivas con Rive cargadas directamente desde la red.

### ♿ Capítulo 11: Accesibilidad y Tema
- **Control de Tema e Inclusividad**: Demostración de cómo usar `Semantics` para lectores de pantalla y un sistema de cambio de tema dinámico (Claro/Oscuro) que respeta los estándares WCAG.

## Requisitos y Configuración

Para ejecutar este proyecto, asegúrate de tener Flutter instalado en tu sistema.

1. Clona el repositorio o descarga los archivos.
2. Navega al directorio del proyecto:
   ```bash
   cd avanzado
   ```
3. Instala las dependencias:
   ```bash
   flutter pub get
   ```
4. Ejecuta la aplicación:
   ```bash
   flutter run
   ```
